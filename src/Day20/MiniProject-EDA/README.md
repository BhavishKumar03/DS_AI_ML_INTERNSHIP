# Customer Analytics EDA Mini Project

This repository contains a Jupyter notebook `MiniProject1_EDA.ipynb` performing exploratory data analysis on the `customer_analytics.csv` dataset. Each record corresponds to a single customer with demographic, income, and behavior features.

## Files

- `customer_analytics.csv` - raw data file provided for analysis
- `MiniProject1_EDA.ipynb` - step-by-step EDA notebook broken into phases: inspection, preprocessing, univariate/bivariate analysis, and multivariate storytelling.
- `requirements.txt` - Python dependencies needed to run the notebook
- `report.md` - concise summary of key insights extracted from the data

## Running the Project

1. Create and activate a Python environment (e.g., `python -m venv venv` then `venv\Scripts\activate`).
2. Install dependencies with `pip install -r requirements.txt`.
3. Start JupyterLab/Notebook in the project folder: `jupyter notebook`.
4. Open `MiniProject1_EDA.ipynb` and run all cells from top to bottom.  

The notebook is self-contained and can be rerun after restarting the kernel.
