# Machine Learning 

Machine Learning is a field of Artificial Intelligence that enables computers to learn patterns from data instead of following explicitly programmed rules. In traditional programming, developers provide both the data and the rules to generate outputs, whereas in Machine Learning the system is given data along with correct outcomes, allowing it to automatically learn the underlying rules and make predictions on new data. Based on how learning occurs, Machine Learning is commonly divided into three types: Supervised Learning, where models learn from labeled data such as predicting house prices or email spam detection; Unsupervised Learning, where models discover hidden patterns without labeled outputs, such as customer segmentation or clustering similar products; and Reinforcement Learning, where an agent learns through trial and error using rewards and penalties, like training robots, self-driving cars, or game-playing AI systems. This data-driven approach makes Machine Learning powerful for solving complex real-world problems where writing explicit rules is difficult.
